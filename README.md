# 株式会社ゆめみ iOS エンジニアコードチェック課題

## 環境
Cocoa pods：
RealmSwift（最新バージョン）
を使用。
起動する際は以下の手順を踏んでください。

1. `pod install`実行
2. finderを開き「iOSEngineerCodeCheck.xcworkspace」をダブルクリック
